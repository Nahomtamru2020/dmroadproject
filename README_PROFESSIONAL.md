# Professional Road Construction Management Platform

## Added Professional Recommendations

This platform is structured for:
- Road construction management
- Contractor reporting
- Daily site supervision
- Machinery tracking
- Personnel management
- Financial reporting
- Timeline tracking

## Deployment

Install dependencies:

npm install

Run locally:

npm run dev

Build production:

npm run build
